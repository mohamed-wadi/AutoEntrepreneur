Si le PC est cassé / formaté : comment revenir au même état
Oui, tu peux revenir au même état données métier (factures, cabinets, formations, etc.) avec les backups DB.

A. Sur la nouvelle machine
Installer Docker + cloner le repo.
Démarrer DB :
docker compose up -d db
Télécharger depuis Drive :
latest.dump (ou un dump daté précis)
latest_uploads.zip (si présent)
B. Restaurer la base
docker cp latest.dump facture-gestion-hub-db-1:/tmp/latest.dump
docker compose exec -T db sh -lc "PGPASSWORD=adminpassword pg_restore -U admin -d facture_db --clean --if-exists /tmp/latest.dump"
C. Restaurer les fichiers uploadés (si tu as latest_uploads.zip)
mkdir .local-uploads
tar -xf latest_uploads.zip -C .local-uploads
(ou extraire avec 7-Zip dans .local-uploads)

D. Lancer l’app
DEMARRER.bat
Ouvrir http://localhost:19509
Tu retrouveras les données frontend/backend puisqu’elles viennent de la DB + fichiers uploadés.
